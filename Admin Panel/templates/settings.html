Settings for {{ server['namecache'] }}

Message parsing

Rosettes will parse messages to extract and provide data for certain things. I.E. Steam games or profiles, mirroring 4cdn uri's, etc. Regardless of this setting, Rosettes NEVER stores any message data whatsoever.

Farm minigame

Toggles the minigame commands under /farm. Be advised, if annoying, this can be limited to a single channel with /setfarmchan.

Gambling commands

Gambling commands are the commands where randomization is relevant. i.e. /dice; /checkem.

Voicechat announce

If enabled, Rosettes will announce when a user joins or leaves a VC, with a message in the channel's text channel. Each message is deleted after 60 seconds, so this does not cause spam.

Additional configuration:

/setlogchan sets a channel where an announcement is made whenever someone joins or leaves the server.
